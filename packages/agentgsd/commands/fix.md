aliases: [debug, fix-bug]
---

Fix the bug in the provided code following systematic debugging:

1. First, understand the bug:
   - Read the relevant code files
   - Identify the expected vs actual behavior
   - Check for error messages or stack traces

2. Reproduce the issue:
   - Create a minimal test case if needed
   - Run the code to confirm the bug

3. Analyze root cause:
   - Trace through the code execution
   - Identify why the bug occurs

4. Implement the fix:
   - Make minimal changes to fix the issue
   - Add comments explaining the fix

5. Verify:
   - Run tests to confirm the fix works
   - Check for any regressions
